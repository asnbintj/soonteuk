• 형제끼리, 자매끼리 페어 구성

⛪ 앱 소개
• 전도(특공) 봉사자들의 구역 배정을 자동으로 생성하는 앱입니다
• 형제끼리·자매끼리 페어(2인 1조)를 구성하고, 각 페어에 고정 구역을 배정합니다
• 전시대는 모든 페어가 라운드로빈으로 균등하게 번갈아 담당합니다
• 배정표와 구역표를 클립보드에 복사해 카톡방에 바로 공유할 수 있습니다
① 교대 시간 선택
상단에서 15분(4교대/시간) 또는 20분(3교대/시간)을 선택합니다.
💡 인원이 많을수록 15분을 권장합니다.
모든 인원이 전시대를 최소 1회 받으려면 타임 수가 충분해야 합니다.
② 봉사자 이름 입력
1
첫 번째 칸(⭐)에 인도자 이름을 입력합니다
2
나머지 칸에 봉사자 이름을 입력하고 형제 또는 자매 버튼을 눌러 성별을 지정합니다
3
인도자와 전시대를 함께 서는 훈련자가 있으면 해당 이름 옆 🎓 훈련 버튼을 누릅니다 (1명만 가능)
4
최소 4명 이상, 최대 10명 이하로 입력합니다
💡 고정 인원 저장을 누르면 자주 함께하는 인원을 저장해두고
다음 번에 불러오기로 빠르게 입력할 수 있습니다.
③ 배정표 생성 & 공유
1
배정표 생성 버튼을 누르면 자동으로 구역이 배정됩니다
2
배정표 상단의 드롭다운으로 구역명을 변경할 수 있습니다
3
🗺️ 구역표를 클립보드로 복사 버튼으로 구역 지도 이미지를 복사합니다
4
📋 배정표를 클립보드로 복사 버튼으로 배정표 이미지를 복사합니다
5
카카오톡 단체방에 붙여넣기(Ctrl+V / 길게 누르기)로 바로 공유합니다
📌 배정 규칙 안내
• 형제끼리, 자매끼리 2인 1조 페어를 구성합니다
• 각 페어는 처음 배정된 구역에 영구 고정됩니다
• 전시대는 모든 페어가 라운드로빈으로 균등하게 번갈아 서게 됩니다
• 홀수 인원일 때는 인도자(⭐)가 배정표에서 제외됩니다
• 🎓 훈련자는 특정 1개 슬롯에서 인도자와 함께 전시대에 배정됩니다
📂 앱 파일 공유 방법
• 이 앱은 단일 HTML 파일로 동작합니다
• 인터넷 없이도 파일을 직접 열어 사용할 수 있습니다
• 카카오톡·밴드 등으로 HTML 파일 자체를 공유하면 누구나 사용 가능합니다
• 입력한 이름·설정은 기기에 자동 저장됩니다
📱 아이패드에서 사파리로 여는 방법
1
카카오톡에서 파일을 꾹 눌러 → 저장 선택 (파일 앱에 저장됨)
2
파일 앱 → 다운로드 폴더에서 HTML 파일을 찾습니다
3
파일을 꾹 누른 후 → "다음으로 열기…" → Safari 선택
4
또는 파일 선택 후 상단 공유 버튼(□↑) → "Safari에서 열기"
💡 파일 앱에서 바로 탭하면 미리보기만 열릴 수 있습니다.
반드시 "다음으로 열기 → Safari" 방법을 사용하세요.
